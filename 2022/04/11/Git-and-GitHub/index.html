Git and GitHub
2022-04-11|
  1. repositorie，仓库，存储代码，它是有地址的

  • 放在仓库的代码，需要通过终端（git clone）保存在本地。

  • markdown，是一种语法

  • 上传自己的代码到仓库：git add，git commit，git push

  • version control：把每次提交的代码记录下来

  1. 新建文件：mkdir 文件夹名称
  2. 命令窗口：cd +文件夹位置；git init；查看此时档案状态：git status；将工作区加入到暂存区：git add .；git commit -m “(操作名称)”；查看更新了几次：git log；
  3. 退出编辑窗口“:wq”，回车。
